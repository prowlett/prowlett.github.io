Linear systems

Peter Rowlett

1 Introduction

Many relationships are linear, and we use linear equations to describe such relationships, e.g. a linear equation of the variables (or ‘unknowns’) \(x_1, x_2, \ldots x_n\) has the form

\[ a_1x_1+a_2x_2+\ldots +a_nx_n=b\text{,} \]

where \(a_1,a_2,\ldots a_n\) and \(b\) are constants.

A linear system is a collection of one or more linear equations for which we seek solutions that satisfy all the equations simultaneously.

2 Matrix notation

A general linear system of \(n\) equations in \(n\) unknowns

can be written as a matrix in the form

\[ \begin{bmatrix} a_{11} & a_{12} & \ldots & a_{1n} \\ a_{21} & a_{22} & \ldots & a_{2n} \\ \vdots & ~ & \ddots & \vdots \\ a_{n1} & a_{n2} & \ldots & a_{nn} \\ \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \\ \vdots \\ x_n \\ \end{bmatrix} = \begin{bmatrix} b_1 \\ b_2 \\ \vdots \\ b_n \\ \end{bmatrix} \]

Write this as \(\mathbf{A}\mathbf{x}=\mathbf{b}\).
 
\(\mathbf{A}\) is called the coefficient matrix.

4 Cramer’s Rule

A Cramer system is any system of \(n\) linear equations in \(n\) unknowns if and only if the matrix formed by the coefficients is non-singular.

Cramer’s Rule (also known as Method of Determinants) makes use of determinants to solve such a non-singular square system.

For a system

\[ \mathbf{A}\mathbf{x} = \mathbf{b}\text{,} \]

where \(\mathbf{x}=\begin{bmatrix} x_1 \\ x_2 \\ \vdots x_n \end{bmatrix}\)

let \(\mathbf{A}_i\) be the matrix obtained by replacing the entries of the \(i\)th column of \(\mathbf{A}\) by the answer vector.

Then

\[ x_i = \frac{\det (\mathbf{A}_i)}{\det (\mathbf{A})}\text{.} \]

Note that for \(\mathbf{A}\mathbf{x}=\mathbf{b}\), if \(\det (\mathbf{A})=0\) then Cramer’s Rule will not work.

5.2 Equivalence of linear systems and matrices

Two linear systems are equivalent if and only if they have the same solution set.

The basic method for solving a system of linear equations is to replace the given system by a new equivalent system that has the same solution set but it is easier to solve.

5.3 Elementary row operations

Transforming one linear system into an equivalent system that is easier to solve uses a series of steps by applying the following three types of operations (known as elementary row operations):

  1. Row switching: A row within the matrix can be switched with another row.

  2. Row multiplication: Each element in a row can be multiplied by a non-zero constant.

  3. Row addition: A row can be replaced by the sum of that row and a multiple of another row.

5.4 Augmented matrix

We can write the system \(\mathbf{A}\mathbf{x}=\mathbf{b}\) as an augmented matrix as

\[ \left[ \mathbf{A}|\mathbf{b} \right] =\left[\begin{array}{cc|c} a_{11} & a_{12} & b_1\\ a_{21} & a_{22} & b_2 \end{array}\right]\text{.} \]

This has the same meaning, it is just a compact notation we can use for row operations.

5.5 Gaussian elimination

Steps:

  1. Write down the augmented matrix corresponding to the given linear system.

  2. Perform a series of elementary row operations to reduce the augmented matrix to an echelon form. This is a matrix that satisfies these conditions:

    1. It is upper triangular (entries below the main diagonal are \(0\));

    2. All zero rows are at the bottom of the matrix;

    3. The leading entry of each non-zero row after the first occurs to the right of the leading entry of the previous row.

  3. Working from the bottom of the echelon matrix upwards, evaluate the unknowns using back-substitution.
