2 Cycle notation

Let \(x_i\) represent a cycle of length \(i\).

Then, for example, we can represent

  • \((1)\) as \(x_1\);

  • \((12)\) as \(x_2\);

  • \((1234)\) as \(x_4\);

  • \((13)(24)\) as \(x_2^2\);

  • \((1)(2)(3)(4)\) as \(x_1^4\);

  • \((1)(2)(34)\) as \(x_1^2x_2\).

In a colouring problem, each member of a cycle must be coloured the same way, since the elements are mapped onto each other by the symmetry in question.

For example, \((13)\) means \( 1 \to 3\) and \(3 \to 1\), so \(1\) and \(3\) must be the same colour.

So if there are \(m\) colours, each \(x_i\) represents a set of elements that can be coloured \(m\) different ways.

3 Burnside’s Theorem

Theorem

For \(G\) a group of permutations of \(\{ 1,2,\dots ,n\} \), the cycle index of \(G\) is

\[ P(x_1,x_2,\dots ,x_n) = \frac{1}{|G|} \sum _{\pi \in G}^{} (\text{cycle representation of $\pi $})\text{.} \]

Then the number of \(m\)-colourings of configurations acted upon by \(G\) is \(P(m,m,\dots ,m)\).

So together these can be coloured \((1+b)(1+g) = 1+b+g+bg\) ways.

There is one element in \(x_1\) and we require that it has a colour, therefore its colours can be expressed as \(b+g\).
